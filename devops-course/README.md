# 🚀 DevOps Course — Guía Completa

> **Nivel de entrada:** Bash avanzado + conocimientos sólidos de redes TCP/IP  
> **Duración estimada:** 12 módulos · ~120 horas de estudio + práctica  
> **Objetivo:** Dominar el stack DevOps completo de forma progresiva y práctica

---

## 🗺️ Mapa del Curso

```
FUNDAMENTOS AVANZADOS
├── Módulo 01 — Bash Scripting Avanzado
└── Módulo 02 — Redes Avanzadas & Protocolos

INFRAESTRUCTURA BASE
├── Módulo 03 — Linux SysAdmin Profundo
├── Módulo 04 — Containers & Docker
└── Módulo 05 — Kubernetes

ENTREGA CONTINUA
├── Módulo 06 — CI/CD Pipelines
├── Módulo 07 — Terraform & IaC
└── Módulo 08 — Ansible & Configuration Management

CLOUD & OPERACIONES
├── Módulo 09 — Cloud (AWS / GCP / Azure)
├── Módulo 10 — Observabilidad & Monitoring
├── Módulo 11 — DevSecOps
└── Módulo 12 — GitOps & Workflow Avanzado
```

---

## 📋 Prerrequisitos

| Habilidad | Nivel esperado |
|-----------|---------------|
| Bash scripting | Loops, funciones, pipes, redirección avanzada |
| Redes TCP/IP | OSI, subnetting, DNS, HTTP/S, TLS |
| Linux CLI | Gestión de procesos, permisos, systemd |
| Git | Branching, merging, rebase básico |
| Programación | Al menos un lenguaje (Python recomendado) |
